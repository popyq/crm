# Spring+SpringMVC+Mybatis的CRM管理系统

它是一个完整的客户关系管理系统，包括市场、销售、服务3大环节，产品成熟，操作简单，功能强大



### 技术栈

Spring+SpringMVC+Mybatis+MybatisPlus+JSP+JQuery+js+ajax+Log4j+Echarts



### 功能模块

- 用户的登录
- 安全退出
- 登录验证
- 创建市场活动
- 分页查询市场活动
- 删除市场活动
- 修改市场活动
- 文件上传导入导入市场活动
- 添加市场活动备注
- 删除市场活动备注
- 线索关联市场活动备注和创建市场活动线索
- 线索转换
- 数据可视化统计图表
- 等
